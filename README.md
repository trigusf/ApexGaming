# ApexGaming 🎮

Modern gaming ecommerce landing page built with Tailwind CSS and JavaScript.  
ApexGaming focuses on creating a modern, responsive, and interactive shopping experience for gaming peripherals such as keyboards, mice, headsets, and accessories.

## ✨ Features

- Modern responsive UI
- Dynamic product rendering using JavaScript
- Best seller products section
- Product category section
- Category-based product counter
- Interactive hover animations
- Reusable card components
- Responsive grid layouts
- Smooth transitions and micro interactions

---

## 🛠️ Tech Stack

- HTML5
- Tailwind CSS
- JavaScript
- DaisyUI
